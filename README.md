Task 3 – SQL Database (E-Commerce Dataset)
This task includes creating a small SQL database for an e-commerce system and running different SQL queries using SQLite (SQLiteOnline).

1. Project Overview
Three tables were created:
customers – customer details
products – product information
orders – purchase records
Sample data was inserted into all tables, and multiple SQL operations were applied.
